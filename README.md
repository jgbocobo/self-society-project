# My Journey of Self-Construction

A personal reflection website exploring identity, growth, and purpose through the lens of self and society.

**Author:** Jorge Bocobo  
**Course:** SAS1-V1, University of the Philippines Los Baños  
**Year:** 2025

## About

This website is a comprehensive exploration of my personal process of self-construction, covering:
- Eight key dimensions of self (perspectives, biology, body image, personality, identity, culture, sexuality, mental health)
- A timeline of life milestones
- Photo gallery showcasing meaningful moments
- Perspectives from friends and family
- Personal and health manifestos

## Live Site

[View the website](#) *(https://github.com/jgbocobo/self-society-project.git)*

## Features

- Fully responsive design (mobile, tablet, desktop)
- Interactive expandable sections
- Auto-playing photo gallery with touch/swipe support
- Smooth scroll animations
- Reading progress indicator
- Modern, clean aesthetic

## Technologies

- HTML5
- CSS3 (with CSS Variables and Flexbox/Grid)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Playfair Display & Inter)

## Local Development

1. Clone this repository
2. Open `index.html` in your browser
3. No build process required!
